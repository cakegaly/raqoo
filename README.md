# raqoo

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/cakegaly/raqoo.git
   ```

2. Navigate to the project directory:

   ```bash
   cd raqoo
   ```

3. Install the dependencies:

   ```bash
   pnpm install
   ```

4. Start the development server:

   ```bash
   pnpm dev
   ```
